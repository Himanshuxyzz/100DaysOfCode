# Day 9
In Day 9 we learn about Types & Numbers in JavaScript.

## JavaScript divide to tow type:
* primitive types:
  * numbers 
  * text or string 
  * Boolean 
  * null
  * undefined

* object types:
  * global object 
  * array 
  * function
  * Classes & Constructors 
  * garbage collection 

## JavaScript Numbers
JavaScript has only one type of number. Numbers can be written with or without decimals.
